# 2D Words Puzzle

## Details
* Title: "Words From Blocks"
* Genre(s): Puzzle 
* Platform(s): PC
* Engine: Unity
* Release: Sep 11, 2024
* Languages: Russian

## Features
* Create words from blocks with letters
* Hint mechanic
* Customizable levels
* Music and SFX
* Words data validator (extended for new languages)
